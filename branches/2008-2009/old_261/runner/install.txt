Перед запуском arch.bat необходимо скопировать исполняемый файл архиватора в директорию, содержащую bat-файл
Для запуска arch.bat необходимо указать 3 аргумента в следующем порядке: 
	тип архива: ar или hf
	директория с набором тестовых файлов
	директория для результатов тестов



Набор тестов находится в директории test:

install.exe 
InstallFSharp.msi 

test1.txt (ранее кодировался)
test2.txt (ранее кодировался с ошибками)
test3.txt (ранее кодировался с ошибками)
test4.txt
test5.txt (файл, содержащий повторяющийся символ)
test6.txt (большой текстовый файл)
one.txt   (содержит 1 символ)

pic1.jpg
pic2.jpg

empty пустой файл
