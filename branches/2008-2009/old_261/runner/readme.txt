Аргументы командной строки:

-help показать описание аргументов
-in [filename] файл для архивации/разархивации
    filename: имя файла
-out [filename] файл для записи результата работы
    filename: имя файла
-action [actiontype] тип действия:
    actiontype: encode - заархивировать
		decode - разархивировать
		test - протестировать архивацию (заархивировать, разархивировать и сравнить результаты)
-algorithm [algname]  алгоритм, по которому должен работать архиватор
    algname: ar - арифметическое кодирование
	     hf - кодирование Хаффмана
-stat показать статистику (коэффициент сжатия)
